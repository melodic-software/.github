# Governance

Melodic Software projects are maintained by the project maintainers.

## Decision Making

Decisions are made by the project maintainers. We welcome community feedback through issues and pull requests.

## Contributions

We operate under a clear set of standards:
- **Trunk-Based Development**
- **Conventional Commits**
- **Automated Testing**

See [CONTRIBUTING.md](CONTRIBUTING.md) for more details.
